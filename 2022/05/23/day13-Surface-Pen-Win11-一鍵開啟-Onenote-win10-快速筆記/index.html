day13-Surface Pen Win11 一鍵開啟 Onenote win10 快速筆記

發表於 2022-05-23 更新於 2022-12-21 分類於 windows

在靈感湧現或者需要複雜推演的場合
Windows上最方便的莫過於Onenote了.
目前找到方法及想法在此紀錄.
windows, win11, surfacepro4

嘗試

畫板(Whiteboard)

  • 因為電腦老, 速度差強人意,
    對於 稍縱即逝 的靈感殺傷力極大
    反而 Win10 的 Whiteboard 畫面簡單速度也快
    果斷放棄

自黏便籤 StickyNote

  • 太小了, 真的是當便條紙在用的.
    但我考量電腦老就不多開這些程序了
    果斷放棄

Onenote 舊版

  • 聽說對於觸控筆支援稍嫌不足
    介面陽春
    果斷放棄

Onenote win10 版

  • 黑介面, 觸控ok
    暫時無可挑剔
    又可以在win10,11中自定按鍵功能
    唯一美中不足
    希望他可以在開啟時自動跳到**快速筆記**

如何開啟時跳轉筆記

確認 onenote-cmd 預設程式

設定->應用程式->預設應用程式->尋框輸入”onenote-cmd”
一般預設就是onenote win10

利用 URI Scheme

URI Scheme 可以在 Run 裡面執行許多 windows指令
詳細可以考 Launch UWP apps via URI Scheme
後來我找到這篇 Launch Onenote UWP using Autohotkey
找到一個方法, 就是新建一個bat檔, 裡面寫入

並且建立一個捷徑, 放置於任意位置, 例如我是放在 C:\

因為 windows設定無法選擇bat檔案, 為此製作一個捷徑檔

如上面圖片顯示, 動作選擇”開啟程式”->選擇程式->選擇該捷徑
大功告成
